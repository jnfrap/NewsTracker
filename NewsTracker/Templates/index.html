NewsTracker

 

NewsTracker is a web scraper that allows you to extract news from a website and save them in a local directory.

 
 

Instructions:

  • Enter the url of the website you want to extract the news from.
  • "Main url" refers to the root url, "News url" refers to the site with the list of news.
  • Enter the tag, type and class of the list of news.
  • Enter the tag, type and class of the title of the news.
  • Enter the tag, type and class of the subtitle of the news.
  • Enter the tag, type and class of the content of the news.
  • Enter the tag, type and class of the images of the news.
  • Enter the name of the directory where you want to save the news.
  • If the title is not correct, it will fail, but if the rest of the fields are wrong it will return nothing.
  • If a tag has several classes, write them all or just one.
  • Tags and attributes are not limited to div and class, they can be anything.
  • The download may take a few minutes. The page is not stuck, give it time to finish.
  • The test button will download the data of the fields without saving internally.
    Again, it can take a few minutes, the page is not stuck, give it time to finish.
